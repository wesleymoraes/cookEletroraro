
# Wave - Boilerplate Com Sass



## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/wesleymoraes/BoilerplatePadrao.git
```

Entre no diretório do projeto

```bash
  cd store-theme
```

Instale as dependências

```bash
  yarn init
```

Rode o yarn

```bash
  yarn
```
inicie o yarn

```bash
  yarn start
```
